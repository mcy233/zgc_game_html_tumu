# 七项目部 · 工地土木模拟器 — 完整玩法指南

> **版本：** v2.0 | **更新日期：** 2026-04-13

---

## 一、游戏概述

这是一款以中国建筑工程行业为背景的**开放式职业成长模拟器**。玩家扮演一名刚进入工地的新人，从见习生做起，通过日常学习、施工管理、团队经营、考证进修等行动，在一个又一个项目中逐步晋升，最终成为项目经理、公司高管，或选择考公、考研、转行IT等人生分支。

**核心玩法循环：** 每季度分配行动点 → 选择行动 → 触发事件 → 季末结算 → 项目完工 → 晋升/调动 → 新项目

|------|------|----------|------|

## 三、季度循环（核心游戏循环）

每个游戏季度（春/夏/秋/冬）的流程：

### 3.1 行动阶段

每季度有 **固定行动次数上限**（由职业等级决定）：

| 等级 | 行动次数/季度 |
|------|--------------|
| 见习（Stage 0） | 18 |
| 正式（Stage 1） | 20 |
| 负责人（Stage 2） | 22 |
| 经理（Stage 3） | 24 |
| 总监/项目经理（Stage 4） | 26 |

每次行动消耗精力、心态、体力中的一项或多项，并产生收益。当任一属性不足以支付消耗时，该行动不可执行。

**除正式行动外，还有免费活动：**
- **工间散步**：每季度最多 3 次，心态+6，体力+6，精力-8
- **宿舍休息**：精力+28，心态-3
- **外出放风**：花费 ¥500，心态+12，体力+10，精力+15

### 3.2 季末结算

季度结束时自动处理：

1. **工序验收**：每个已提交的工序段有 **52%** 概率通过
   - 通过：心态+12，声望+6，领导信任+4，完工段数+1，经验+15，扣 ¥300-499（请监理吃饭）
   - 未通过：心态-10，声望-3，体力-3，精力-6，领导信任-2

2. **工资发放**：按当前职级的基本工资发放一次
   - 甲方满意度 ≥90 → 额外 +¥2,000
   - 甲方满意度 ≥70 → 额外 +¥1,000
   - 甲方满意度 ≥50 → 额外 +¥500
   - 甲方满意度 <30 → 倒扣 -¥500

3. **属性变化**：
   - 精力重置为 100
   - 心态 +8（自然恢复）
   - 体力 +5（自然恢复）
   - 经验 +5（自然积累）
   - 材料 +400（基础补给）+ 资产产出

4. **衰减机制**：
   - 若本季度没有找领导汇报 → 领导信任 **-5**
   - 若本季度没有与某同事互动 → 该同事好感 **-4**

5. **甲方满意度变化**：
   - 进度 >60%：+2；>80%：再+3
   - 安全隐患 >50：-3；>80：再-5
   - 每个完工段：+1
   - 声望 >60：+1

6. **随机事件**：35% 概率触发季度事件（见第八章）

### 3.3 天气系统

每季度随机生成天气，影响施工类行动：

**季节天气概率池：**
| 季节 | 可能天气 |
|------|---------|
| 春季 | 小雨(多)、多云(多)、晴、大雨、大风 |
| 夏季 | **高温(多)**、大雨(多)、晴、多云 |
| 秋季 | **晴(多)**、多云(多)、大风、小雨 |
| 冬季 | **寒潮(多)**、多云、大风、晴、小雨 |

**天气对施工进度的影响倍率：**
| 天气 | 进度倍率 | 体力消耗倍率（仅室外行动） |
|------|---------|--------------------------|
| 晴 | ×1.1 | ×1.0 |
| 多云 | ×1.0 | ×1.0 |
| 小雨 | ×0.85 | ×1.1 |
| 大雨 | ×0.6 | ×1.3 |
| 高温 | ×0.75 | ×1.4 |
| 寒潮 | ×0.70 | ×1.3 |
| 大风 | ×0.80 | ×1.15 |

> **室外行动** = 盯施工、现场巡检（体力消耗受天气额外加成）

## 六、项目系统

### 6.1 项目类型

| 类型 | 基础工期 | 浮动 | 难度 | 声望倍率 | 解锁条件 |
|------|---------|------|------|---------|---------|
| 住宅楼盘 | 8季度 | ±1 | ×1.0 | ×1.0 | 起始可选 |
| 商业综合体 | 10季度 | ±2 | ×1.3 | ×1.3 | 第2个项目+, Stage≥1 |
| 市政桥梁 | 10季度 | ±2 | ×1.5 | ×1.5 | 第3个项目+, Stage≥1 |
| 超高层 | 12季度 | ±2 | ×1.8 | ×1.8 | 第3个项目+, Stage≥2 |
| 隧道/地铁 | 12季度 | ±3 | ×2.0 | ×2.0 | 第4个项目+, Stage≥2 |

### 6.2 项目阶段

每个项目按工期比例划分为四个阶段：

1. **进场准备（PREP）** — 约15-20%工期
2. **基础施工（FOUNDATION）** — 约20-30%工期
3. **主体施工（MAIN）** — 约35-45%工期
4. **收尾验收（FINISHING）** — 约15-25%工期

阶段切换时进度归零重新计算。

### 6.3 工序提交机制

通过**写施工方案**和**盯施工**行动积累方案进度（planProgress），当方案进度 ≥100 时自动提交一个工序段，方案进度归零。

提交的工序在季末结算时有 52% 概率通过验收。

### 6.4 项目完工评分

项目工期到达后进行评分：

- **进度分**：最终进度 + 完工段数 × 5（满100）
- **安全分**：100 - 安全隐患值
- **甲方满意分**：甲方满意度原值
- **领导评价分**：领导信任度原值

四项平均得出综合评级：

| 平均分 | 评级 | 经验倍率 | 奖金倍率 |
|--------|------|---------|---------|
| ≥85 | S | ×1.5 | ×1.5 |
| ≥70 | A | ×1.2 | ×1.2 |
| ≥55 | B | ×1.0 | ×1.0 |
| ≥40 | C | ×0.7 | ×0.7 |
| <40 | D | ×0.4 | ×0.4 |

**完工奖励公式：**
- 经验 = 60 × 难度倍率 × 评级倍率 × (1 + 职级 × 0.3)
- 声望 = 8 × 声望倍率 × 评级倍率
- 奖金 = ¥5,000 × 评级倍率 × 难度倍率
- 人脉 = 5 × 评级倍率 + 职级 × 2

### 6.5 项目调动

完工后进入调动期：
- 部分属性恢复（精力+30, 体力+20, 心态+15）
- 安全隐患降低10点
- 好感≥65的同事有30%概率跟随到新项目
- 可选择新项目类型（根据职级和项目序号解锁）

### 6.6 领导类型

每个项目随机分配领导类型，影响收益倍率：

| 类型 | 说明 | 影响 |
|------|------|------|
| MICROMANAGER | 事必躬亲型 | 进度、信任收益受到影响 |
| GHOST | 放任型 | 不太管你，但信任也不太涨 |
| SUPPORTIVE | 支持型 | 进度和信任收益更高 |
| CELEBRITY | 明星领导 | 特殊互动模式 |

> **领导信任 >80 时**：所有行动的心态消耗 ×0.7（打折）

## 八、随机事件系统

### 8.1 季度事件

每季度末有 **35%** 概率触发一个随机事件，事件内容根据当前项目阶段加权：
- 60% 概率从**当前阶段事件池**抽取
- 40% 概率从**通用事件池**抽取

每个事件提供 2-3 个选项，每个选项有不同的属性变化（进度、心态、安全、金钱等）。

### 8.2 通用事件示例

- **材料短缺**：选择高价采购 / 等待 / 找关系
- **设计变更**：接受 / 谈判 / 拒绝
- **分包商捷径**：同意冒险 / 拒绝 / 上报
- **暴风来袭**：停工 / 赶工 / 部分保护

### 8.3 阶段专属事件

不同项目阶段有专属事件池：
- **进场准备**：场地问题、手续办理等
- **基础施工**：地质情况、基坑风险等
- **主体施工**：工期压力、质量检查等
- **收尾验收**：验收整改、结算纠纷等

## 九、资产系统

通过游戏过程中的随机资产推荐（`pendingAssetOffer`），可以购买资产。资产提供持续性被动加成：

- **材料产出**：每季度额外获得材料
- **工资产出**：每季度额外获得工资
- **心态/体力消耗减免**：降低行动的属性消耗
- **进度加成**：提升行动的进度收益

资产效果可叠加。

## 十一、游戏结束条件

### 11.1 负面结局（Game Over）

|---------|---------|------|
| 心态 ≤ 0 | 精神崩溃 | 长期高压导致心理问题 |
| 体力 ≤ 0 | 体力不支 | 过劳导致身体崩溃 |
| 精力 ≤ 0 | 精力枯竭 | 连续高强度行动耗尽 |
| 声望 ≤ 0 | 声誉破产 | 口碑崩塌，行业混不下去 |
| 安全隐患 ≥ 100 | 安全事故 | 重大安全事故发生 |

### 11.2 正面结局

通过人生选择触发（见第七章），根据积累的各项数值评定结局等级。

## 十二、核心策略建议

### 12.1 新手期（Stage 0）

- **优先安全培训**：降低安全风险，同时积累证书为晋升做准备
- **利用"当资料员"赚钱**：Stage 1 后该行动消失
- **岗位专属行动经验值极高**：跟师傅学/跟商务对量/核对试验曲线 都给 7-8 经验
- **注意心态管理**：现场巡检可恢复心态12点

### 12.2 成长期（Stage 1）

- **盯施工 + 写方案**是进度双引擎
- **写方案有 25% 概率增加安全隐患**（+3~7），需要用安全培训或巡检对冲
- **找领导汇报**是信任核心来源（+15），不汇报每季度自动-5
- **开始考证**：Stage 2 需要证书≥20

### 12.3 领导期（Stage 2+）

- **组织专家论证**（BUILD）是性价比最高的高阶行动（经验+10, 进度+10）
- **带新人**低消耗且恢复心态
- **关注人脉积累**：Stage 3 需要人脉≥40

### 12.4 人生路线规划

- **考公路线**：从 Stage 1 开始考公刷题，需要攒够 20 点积累 + 证书20 + 余额5万
- **转行IT**：任何时候都可以编程自学，需要 15 技能 + 人脉50 + 余额3万
- **考研路线**：仅 Stage ≤1 可选，需要 15 积累 + 证书15
- **创业路线**：Stage≥2, 余额10万+, 人脉60+

### 12.5 天气应对

- **大雨/寒潮天气**不要安排施工类行动（进度仅 ×0.6-0.7）
- **晴天抢施工**（进度 ×1.1）
- 恶劣天气适合做学习、社交、室内类行动
