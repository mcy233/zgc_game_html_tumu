# 七项目部 · 工地土木模拟器

> 重生之我在工地做土木 —— 一款以中国建筑工程行业为背景的**开放式职业成长模拟器**。

你是一名刚踏入工地的见习生。安全帽一戴，板房一住，从此开始在项目部打拼的漫长征途。考证、盯施工、写方案、跟领导汇报、请客吃饭、甚至偷偷考公考研……在**心态、体力、精力、安全隐患与声望**之间求生存，在一个又一个项目中逐步晋升，最终成为项目经理——或者选择一条完全不同的人生道路。

## 游戏特色

- **三条职业路线**：技术岗 / 施工岗 / 商务岗，起点不同，路径不同
- **五级晋升体系**：见习 → 正式 → 负责人 → 经理 → 总监/项目经理
- **五种项目类型**：住宅楼盘、商业综合体、市政桥梁、超高层、隧道/地铁
- **30 个行动选择**：学习、工作、团队三大面板，每个行动附带策略提示
- **动态天气系统**：晴天加速施工，大雨暴雪减缓进度，Canvas 实时渲染
- **季度事件系统**：随机事件按项目阶段加权，每个选择影响全局走向
- **人生分支**：考公上岸、转行互联网、考研深造、分包创业……7 条人生路线
- **50+ 成就称号**：从「工地小白」到「行业大佬」，隐藏成就等你发掘
- **明暗主题切换**：支持亮色/暗色/跟随系统三种模式
- **迷你游戏**：混凝土振捣、安全隐患排查、测量瞄准等 5 款嵌入式小游戏

## 在线体验

推送 `main` 分支后，由 GitHub Actions 自动构建部署到 GitHub Pages。

1. 仓库 **Settings → Pages**：Source 选 **GitHub Actions**
2. 部署完成后访问：`https://<你的用户名>.github.io/zgc_game_html_tumu/`

## 本地运行

```bash
npm ci
npm run dev
```

浏览器访问 `http://localhost:5173/` 即可开始游戏。

## 技术栈

- **React 19** + **TypeScript**
- **Vite 6** 构建
- **Tailwind CSS 4** 样式
- **Canvas API** 天气粒子渲染
- **Web Audio API** 音效系统

## 项目结构

```
src/
├── app/            # 游戏流程辅助
├── components/     # UI 组件（布局/面板/弹窗/卡片）
├── config/         # 游戏配置与初始状态
├── data/           # 行动/项目/事件/成就/人物数据
├── engine/         # 核心引擎（行动处理/季度结算/晋升/天气）
├── hooks/          # React Hooks（状态管理/操作处理）
├── systems/        # 系统模块（动画/音效/迷你游戏/主题）
├── types/          # TypeScript 类型定义
└── utils/          # 工具函数
docs/
├── 游戏完整玩法指南.md    # 详细玩法文档
└── 策划案.md              # 游戏设计文档
```
